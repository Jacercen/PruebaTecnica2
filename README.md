# PruebaTecnica2
Prueba Técnica - Curso 2
Segunda Pruba técnica del curso, la cual es un programa para filtrar vuelos,
en este caso las fechas ya vienen daddas por defecto y los vuelos tambien, el programa
te permite filtrar estos vuelos segun una fos fechas, fechaInicio y fechaFin, las cuales se
comparan con la fecha de salida del vuelo.
El programa cuenta con un Main que es el que debemos ejecutar para comprobar el funcionamiento
mediante la salida de texto por consola, cuenta tambien con un entities-Vuelo.java que es el encargado
de gestionar el objeto Vuelo con sus atributos, un servicies-VueloServicio.java que es el encargado de gestionar
los distintos métodos que consideramos reutilizables para el programa como puede ser mostrar los vuelos por
pantalla y un utils-FiltroVuelo.java donde tenemos la clase utilitaria para el filtro de los vuelos.

En el programa se usan las clases:
  - java.time.LocalData
  - java.util.HashMap
  - java.util.Map
  - java.util.List
  - java.util.Optional
  - java.util.ArrayList
